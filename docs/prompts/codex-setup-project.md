# Codex Prompt: Initialize Project

Use this prompt in Codex after creating an empty repository.

```text
You are working in a TypeScript React Vite repository for a course project called Task Tracker.

Read AGENTS.md before making changes.

Initialize the application using:
- TypeScript
- React
- Vite
- Vitest

Preserve the documented repository structure.

Create the initial src/features/tasks folders:
- domain
- strategies
- services
- repositories
- __tests__

Do not implement full UI yet. Only create the project scaffold and ensure tests can run.

After implementation:
- list changed files,
- explain how AGENTS.md was followed,
- provide the commands to run the app and tests.
```
